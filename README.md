APEX HYBRID COMMAND [v4.5.1]
This is a tactical network dashboard developed to simulate real-time handover logic between LTE (High-Speed) and UHF (Long-Range) frequency bands. It features dynamic throughput gauging, a visual navigation assistant for signal recovery, and an integrated COMMS_AI module for live technical advisories.

Key Technical Features:

Dual-Tower Simulation: Diagonal LTE base station deployment with 10km radius thresholds.

Performance Analytics: Real-time Speed vs. Distance trace mapping via Recharts.

Failover Logic: Automatic protocol switching to UHF when out of range to maintain critical data transfer.
